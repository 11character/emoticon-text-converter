# Project Status: Emoticon Text Converter

이 문서는 프로젝트의 개발 과정과 현재 상태를 기록하는 대시보드입니다.

## 1. 프로젝트 개요 (Overview)
- **목적**: 타입 안전성을 갖춘 순수 자바스크립트 기반의 범용 이모티콘 변환 라이브러리 구축.
- **핵심 목표**: 실시간 HTML 변환, 정밀한 커서 위치 유지, 이벤트 기반 인터페이스 제공.

## 2. 모듈 현황 (Module Status)

| 모듈 (Vanilla JS/TS) | 주요 역할 | 상태 |
| :--- | :--- | :--- |
| `src/parser/TextParser.ts` | 텍스트 <-> HTML 변환 로직 | 완료 |
| `src/utils/CursorManager.ts` | TreeWalker 기반 커서 관리 | 완료 |
| `src/core/EmoticonTextConverter.ts` | 이벤트 바인딩 및 상태 관리 | 완료 |

## 3. 현재 진행 상황 (Current Progress)
- [x] **프로젝트 초기화**: ESM 모듈 구조 및 테스트 환경(Vitest, JSDOM) 구축.
- [x] **TextParser 구현**: 특수문자 엔티티화 및 정규식 기반 키워드 변환 로직 완성.
- [x] **CursorManager 구현**: `TreeWalker`를 이용한 논리적 커서 오프셋 관리 로직 완성.
- [x] **Core Integration**: 메인 클래스 통합 및 상태 관리 로직 적용.
- [x] **TypeScript 전환**: 프로젝트 전체 타입스크립트 마이그레이션 및 타입 안전성 확보.
- [x] **기능 정교화**: 엔터 키 동작 제어, 콜백 기반 이벤트 시스템(onInput, onEnter 등) 전환 완료.
- [x] **권한 시스템 구축**: 객체 기반 그룹(`allowedGroups`) 시스템을 통한 $O(1)$ 검색 최적화.
- [x] **커서 정밀도 강화**: `Range` API 기반 커서 위치 계산 및 동기식 위치 보정 로직 통합.
- [x] **실시간 옵션 반영**: `setOptions` 호출 시 파서 재초기화 및 에디터 내용 즉시 갱신 로직 추가.
- [x] **검증 및 테스트**: 엣지 케이스 포함 총 40개 테스트 케이스 통과.
- [x] **데모 및 문서화**: 시뮬레이터가 포함된 데모 페이지 및 사용 가이드 작성.

## 4. 핵심 기술 노트 (Technical Notes)
- **커서 관리**: `img`(이모티콘)와 `br`을 각각 1글자로 취급하는 논리적 위치 시스템.
- **이벤트 안전성**: 사용자 입력과 내부 변환 작업을 분리하여 무한 루프 및 충돌 방지.
- **텍스트 보정**: 이모티콘 삽입 시 이미지 태그 길이를 실제 키워드 길이로 자동 치환.
- **그룹 권한**: `allowedGroups` 설정을 통해 사용자별 이모티콘 노출 권한을 유연하게 제어.
